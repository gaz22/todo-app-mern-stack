# todo-app-mern-stack<br/>
Basic ToDo App using MERN stack (MongoDB, Express, React, and Node.js)<br/>
ToDo list is sorted from High - Low <br/>
// running backend server <br/>
cd backend<br/>
npm install<br/>
nodemon server<br/>

// running front-end server <br/>
cd..<br/>
cd frontend<br/>
npm install<br/>
npm start<br/>
